What This Project Demonstrates

Three core pillars — all implemented manually in Python, no external crypto libraries

🔒

AES-256 Engine

Manual block-by-block Advanced Encryption Standard (AES) with 14 rounds of transformation (SubBytes, ShiftRows, MixColumns, AddRoundKey). Runs CBC mode with a fresh random IV per encryption.

👤

Secure Authentication

User registration and login with SHA-256 + random salt password hashing, brute-force lockout after repeated failures, and OTP-based 2-Factor Verification via email for every login attempt.

🗄️

AES-256 Encrypted Storage

User notes are encrypted with AES-256 CBC before being persisted to an SQLite database. Each note gets a fresh cryptographically random 256-bit key. Plaintext never touches the disk — data is decrypted only on authenticated read requests.
